Jak skompilować?

Najprostszym sposobem aby uruchomić program jest otworzenie go w jakimś środowisku informatycznym, które wspiera język Python typu Pycharm lub VSCode.

W przypadku nie posiadania podanych wyżej programów należy wypakować plik w dowolne miejsce, a potem z poziomu konsoli (wiersza polecenia lub np. Windows Power Shell) znajdując się w katalogu do którego wypakowaliśmy dany plik uruchomić go komendą: python3 main.py.

Do uruchomienia programu potrzebny jest Python w wersji 3.0 lub wyższej, a także biblioteka numpy, którą można zainstalować wpisując w wierszu poleceń: pip install numpy. W przypadku nie posiadania żadnej wersji Pythona na komputerze najlepiej będzie skorzystać z opcji alternatywnej.

Alternatywną opcją jest przeklejenie zawartości pliku main.py (otwierając go w dowolnym edytorze tekstowym) do kompilera online np.: https://www.programiz.com/python-programming/online-compiler/ i uruchomienie pliku.

Co zwraca program?

Program powinien wypisywać wartości wyrazów o numerach 1, 5, 10, 15, 20, 25 oraz 30 dla trzech typów danych obliczanych na dwa sposoby (związkiem podanym w zadaniu oraz związkiem zmodyfikowanym). Dla pierwszego sposobu program powinien wyrzucić trzy błędy, sygnalizujące próbę dzielenia przez liczbę o bardzo małej wartości (~=0).
